Este é um site para análise de dados e cálculo de probabilidade de aprovação para cursos superiores através do SISU.


Como funciona?

Para utilizar o site, selecione a categoria de interesse no menu acima.

Na página de análise, você poderá visualizar algumas informações relacionadas ao SISU 2022.1, geradas a partir dos dados mais recentes referentes ao programa. Como complemento, você pode obter resultados específicos aplicando os filtros disponíveis na parte superior do dashboard.

Na aba de predições, será necessário que você informe os dados solicitados para que o sistema calcule a probabilidade de aprovação para o curso e universidade selecionados.
