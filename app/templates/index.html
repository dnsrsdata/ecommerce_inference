Seja bem-vindo(a)!

Este é um site para análise de dados e cálculo de probabilidade de aprovação para cursos superiores através do SISU.


Como funciona?

Para utilizar o site, selecione a categoria de interesse no menu acima. Na página de análise, você poderá selecionar o curso e a universidade de interesse e visualizar os dados relacionados a eles. Como os dados do SISU 2023 ainda não foram divulgados até a data de publicação deste site, os resultados apresentados são referentes ao SISU 2022. Na página de predições, será necessário que você informe os dados solicitados para que o sistema calcule a probabilidade de aprovação para o curso e universidade selecionados.
